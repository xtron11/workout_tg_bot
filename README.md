# 🏋️ Workout Bot

Телеграм бот для записи на тренировки. Доступ только по одобрению тренера.

## 📋 Функционал

### Для пользователя:
- Запись на тренировку с выбором дня и времени
- Перезапись и отмена записи
- Просмотр своих записей
- Напоминание за день до тренировки
- Защита от двойной записи на один день

### Для администратора:
- Одобрение новых пользователей
- Добавление слотов с выбором дня и времени
- Просмотр всех записей по дням
- Удаление слотов и записей пользователей
- Управление пользователями
- Уведомления о новых записях и отменах
- Рассылка уведомлений всем пользователям

## 🛠 Технологии
- Python 3.11
- Aiogram 3.x
- PostgreSQL
- SQLAlchemy (asyncpg)
- APScheduler
- Pydantic Settings

## 🚀 Установка

1. Клонируй репозиторий
2. Создай виртуальное окружение: python -m venv venv
3. Активируй: venv\Scripts\activate (Windows) или source venv/bin/activate (Mac)
4. Установи зависимости: pip install -r requirements.txt
5. Создай .env файл
6. Запусти бота: python main.py

## ⚙️ .env файл
```
DB_HOST=
DB_PORT=
DB_USER=
DB_PASS=
DB_NAME=
BOT_TOKEN=

```
